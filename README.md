# 🌐 EduCenter — Web-sayt

O'quv markazi uchun Flask web-sayt.

## Ishga tushirish

```bash
pip install -r requirements.txt
python app.py
```

Sayt: `http://127.0.0.1:5000`

## Admin kirish
- Telefon: `admin`
- Parol: `admin123`

## Fayllar

- `app.py` — asosiy fayl, barcha routelar
- `forms.py` — login, register, kurs, o'qituvchi formalari
- `models.py` — SQLAlchemy modellari
- `requirements.txt` — kutubxonalar
- `templates/` — HTML shablonlar
- `static/` — CSS va rasmlar
